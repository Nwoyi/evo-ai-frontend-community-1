## 3. Protected Visual Identity Elements (Brand Assets)

The following are the "Protected Visual Identity Elements":
(a) Product name and mark: "Hyvelox";
(b) Official logo: the three-hexagon icon (two teal hexagons + one coral
    hexagon containing a stylised growth arrow);
(c) Official wordmark: "Hyvelox" set in the official typography;
(d) Copyright line: "© 2026 Hyvelox";
(e) Official palette:
    Primary teal:   #1F7A8C
    Accent coral:   #E37753
    (Additional surface and text tokens are defined in the design system.)
(f) Typography: ui sans serif, system ui, sans serif as the in-app default.

## 4. Restrictions for forks, modified versions, and redistributions

4.1. **No modification of Protected Visual Identity Elements when using
Hyvelox brand assets.** If you distribute or publicly host any version of the
user interface that uses Hyvelox brand assets, you may not remove, replace,
modify, disable, or bypass any Protected Visual Identity Element.

4.2. **Modified UI must remove Hyvelox brand assets.**
If you distribute or publicly host a modified user interface or user experience
(including changes to branding or visual identity), you must:
(a) remove and replace all Hyvelox trademarks and brand assets from the user
    interface, product name display, and related marketing materials for that
    distribution; and
(b) choose a name and branding that are clearly distinct from Hyvelox.

4.3. **No endorsement.**
You may not use Hyvelox trademarks or brand assets in a way that implies
Hyvelox sponsors, endorses, or is affiliated with your distribution, except for
unmodified official builds.

4.4. **No derivative works of the logo and wordmark.**
You may not modify, adapt, recolor, redesign, or create derivative works of the
Hyvelox logo or wordmark.
